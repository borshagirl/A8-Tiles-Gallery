***Tiles Gallery***

## Project Purpose
Tiles Gallery is a modern web application which doing tiles/products showcase. User authentication with (email/password + Google) and user logining/signup and managing their profile.

**Github Repo**
https://github.com/borshagirl/A8-Tiles-Gallery


**Key Features**

* Authentication system (Better Auth + MongoDB)
* Email & Password Sign Up / Sign In
* Google Social Login
* User Profile Management
* User Avatar/Image Support
* Global Loading UI
* Custom 404 Not Found Page
* Toast Notification (react-toastify)
* Responsive & Modern UI (HeroUI + Tailwind CSS)


***Technologies Used***

* Next.js 16 (App Router + Turbopack)
* Tailwind CSS
* HeroUI (UI Components)
* Better Auth
* MongoDB
* React Toastify
* React Icons
* Gravity UI Icons
* Marquee
